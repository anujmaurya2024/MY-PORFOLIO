# Full-Stack Portfolio

This is a full-stack portfolio application built with React, Node.js, Express, and MongoDB.

## Quick Start

To run the entire project (backend and frontend), simply run the included batch file:

1.  Open the project folder: `portfolio`
2.  Double-click on **`START_PROJECT.bat`**

This script will:
- Install all necessary dependencies for both frontend and backend.
- Start the backend server on `http://localhost:5000`.
- Start the frontend development server on `http://localhost:5173`.

## Requirements

- **Node.js** and **npm** must be installed on your system.
- **MongoDB** should be running locally (or update the `MONGODB_URI` in `backend/.env`).

## Manual Setup

If you prefer to run things manually:

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
